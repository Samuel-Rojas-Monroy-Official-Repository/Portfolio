Plataforma de gestión de torneos deportivos en tiempo real
React TypeScript Vite Supabase PostgreSQL Realtime Tailwind CSS PWA

PWA para la gestión integral de torneos de fútbol amateur. Administradores controlan partidos y estadísticas en tiempo real, mientras jugadores e invitados consultan información al instante desde cualquier dispositivo.

⚡ Realtime bidireccional Cambios del admin se reflejan instantáneamente en todos los dispositivos usando Supabase Realtime.
🔐 Auth por roles Tres niveles (admin, jugador, invitado) con políticas RLS en PostgreSQL.
🕸️ Diagrama de araña SVG Visualización de habilidades generada matemáticamente sin librerías externas.
📱 PWA mobile-first Diseñada para usarse en cancha desde el celular, instalable en pantalla de inicio.
